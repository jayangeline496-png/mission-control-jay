# Mission Control

A high-performance project tracker and mission control center for Jay Angeline's worldbuilding and content pipeline.

## Tech Stack (Proposed)
- **Frontend:** Streamlit or Next.js (React)
- **Backend:** Python (FastAPI) or Node.js
- **Data Source:** Google Sheets API (Refactor Tracker) & GitHub API (Drafts Archive)
- **Deployment:** GitHub Pages (Static) or Streamlit Cloud / Vercel (Dynamic)

## Goals
- Visual Kanban board for blog refactors.
- "Mission Dashboard" for daily strategic priorities.
- One-click task assignment for Jay (the AI).
